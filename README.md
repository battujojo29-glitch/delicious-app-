## ✨ Features

- 🟣 **Splash Screen** — Bold branded entry with a full-screen hero image and CTA
- 🛒 **Menu Browsing** — Grid layout with food cards, prices, favourites & add-to-cart
- 📄 **Item Detail** — Full image, recipe info, delivery location & estimated time
- ➕ **Cart Management** — Quantity selector with live total update
- 🟢 **Persistent Cart Bar** — Always-visible cart summary at the bottom of every screen
